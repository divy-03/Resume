\section*{Projects}

\textbf{KickBid} 
(\href{https://github.com/divy-03/KickBid-client-js}{GitHub} / 
\href{https://kickbid.onrender.com}{Live}) \\
A full-scale tournament auction platform designed for campus sports events. Built an end-to-end system where players register, receive ratings, join auctions, and get assigned to teams through live bidding. Built real-time dashboards for auction progress, team composition, and player stats, along with an admin panel for managing events, approvals, and overall tournament flow. Developed complete authentication, role-based access, API integrations, and optimized the backend for high-traffic auction events. \\
\textit{Tech: React, Sass, Node.js, Express, MongoDB, RTK Query}

\textbf{Carido — AI-Powered Car Marketplace} 
(\href{https://github.com/divy-03/car-app}{GitHub} /
\href{https://carido.vercel.com}{Live}) \\
An AI-integrated car marketplace enabling users to list, search, and explore cars with AI-generated details. Implemented server-side filtering, dynamic search, user roles, and secure listing workflows using Prisma ORM with PostgreSQL. Integrated Gemini API for auto-filling car descriptions, and ImageKit for optimized image storage and delivery. Features a clean UI, performance-focused API routes, and scalable architecture deployed on Vercel + NeonDB. \\
\textit{Tech: Next.js 15, TypeScript, Prisma, PostgreSQL, Tailwind, Zustand, ImageKit, Docker}

\textbf{JSON-RPC Load Balancer} (\href{https://github.com/divy-03}{GitHub}) \\
A distributed load-balancing system built to handle JSON-RPC based microservice calls. Implemented intelligent request routing, retry mechanisms, fault detection, and health checks to ensure high availability across multiple backend nodes. Added Redis caching and queueing to improve throughput and reduce latency. Designed the system following fundamental distributed systems principles such as failover, horizontal scaling, and graceful degradation. \\
\textit{Tech: Node.js, Redis, JSON-RPC, System Design Basics}

\textbf{DocAI — Intelligent Document Extraction Engine} \\
A modular document processing system that extracts structured information using LLMs. Built a backend pipeline to process PDFs/images, clean content, and pass them through AI models for entity extraction. Created a frontend dashboard for uploading files, viewing parsed data, and exporting results. Focused on reliability, prompt-tuning, and response validation to make the system practical for real use cases. \\
\textit{Tech: Node.js, Express, Next.js, AI APIs}
