\section*{Projects}


\textbf{KickBid} (\href{https://github.com/divy-03/KickBid-client-js}{GitHub}/\href{https://kickbid.onrender.com}{Live}): A Web platform to facilitate sports tournaments. \\
KichBid will allow students to register as players, be rated based on their skills, and participate in an auction in which captains bid to form their teams. The platform provides real-time updates, team statistics, and tournament schedules, enhancing the experience for both players and spectators. It also offers an admin dashboard for managing events and ratings. \\
\textit{Technologies: HTML, CSS, Sass, React, JavaScript, MongoDB, Express, Node, RTK-Query}

\textbf{Leaf} (\href{https://github.com/divy-03/Leafront}{GitHub}/\href{https://leafront.vercel.com}{Live}): Client-side application for the Leafman back-end system \\
Developed a modern Next-based front-end to serve as the client for the \href{https://github.com/nightmare-tech/leafman}{leafman} back-end, implementing robust state management and seamless API integration using RTK Query. The project features a leaf-inspired loader, responsive UI design, and smooth animations powered by Framer Motion, ensuring both usability and aesthetics.\\
\textit{Technologies: NextJs, TypeScript, RTK Query, Tailwind CSS, Framer Motion, CSS}

\textbf{Carido}(\href{https://github.com/divy-03/car-app}{GitHub}/\href{https://carido.vercel.com}{Live}): AI-Powered Car Marketplace Web App \\
A full-stack car listing platform built with Next.js 15, Tailwind CSS, Prisma, and PostgreSQL. Integrated the Gemini API to enable AI-driven car search and auto-generate car details during listings. Features include advanced filtering, detailed car pages, image generation \& uploads via ImageKit, and seamless seller-buyer interaction. Deployed on Vercel with Prisma-managed Postgres (Neon) database. \\
\textit{Technologies: NextJs, Prisma, PostgreSQL, Tailwind CSS, ImageKit, Docker, NeonDB, Zustand, TypeScript, NextAuth, Zod }

\textbf{Personal Blog Website}(\href{https://github.com/divy-03/blog-website}{GitHub}/\href{https://blog-excl.netlify.app}{Live}): Static Blog Powered by Hugo \\
A personal blogging platform built with Hugo and customized using the Reimu theme. Tweaked layouts, colors, and assets to create a clean and minimal look for publishing Markdown-based posts. The site supports easy content management through Markdown files, auto-generates static pages, and is optimized for fast loading. Deployed seamlessly on Netlify with continuous deployment from GitHub. \\
\textit{Technologies: Hugo, Markdown, Netlify, Git, Static Site Generation }
